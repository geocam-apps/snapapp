This is a fresh project which a user will be using to develop a web app. 

The user is likely to be a non-developer, who is building one-off apps to solve problems using our infrastructure and with access to their geospatial data we host for them

The user probably doesn't know about git, or if they do only have heard about it. 

Manage commits, branches, rollbacks etc on their behalf unless they specify otherwise. 

There is an S3 datastore and an NFS mount to store persistent data. This container has no persistent filesystem
